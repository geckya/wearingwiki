Half Front Wrap Cross Carry

abbreviation HFWCC, Half FWCC
type front carry
pass types used Torso Pass, Cross Pass
size Base Minus 3
finish variations
number of shoulders 2
level beginner
type of wrap Woven Wrap
characteristics
good for Straighteners, Nursing, Sleep Transfer, Toddlers and Support

Carry Recipe

Half Front Wrap Cross Carry – Base Size Minus 3 – Torso Pass + Cross Pass

  • Short tail over one shoulder and across back diagonally
  • Long tail under arm, around belly, and across back for a Torso Pass and then over shoulder
  • Baby goes in pocket
  • Long tail comes down for Cross Pass under baby’s leg
  • Tie off with Slip Knot
